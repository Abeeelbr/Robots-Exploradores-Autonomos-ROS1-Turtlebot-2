Paquete de Exploración
======================

Este paquete contiene los servidores de acción necesarios para la detección y navegación.

**Módulos:**

- **squad_object_detection_action.py:** Detección de objetos en imágenes RGBD.
- **squad_approach_control_action.py:** Aproximación a objetos detectados.
- **squad_autonomous_control_action.py:** Navegación autónoma evitando obstáculos.
